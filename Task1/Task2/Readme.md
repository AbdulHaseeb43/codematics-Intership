Catchub Landing Page
Project Overview
This project is a responsive landing page for the Catchub application, developed using HTML5 and CSS3. The page presents the core features of the Catchub platform, including Channels, Messages, and Future Features, along with app download options.

The implementation focuses on a clean UI, consistent layout, and responsiveness across different screen sizes while maintaining visual balance between text and images.

Objectives
Create a modern landing page for a mobile application
Implement a responsive layout for desktop and mobile devices
Present application features in a clear, structured format
Maintain clean and readable HTML and CSS code
Features Implemented
Responsive Design

Adapts smoothly to different screen sizes using CSS media queries
Optimized layout for both desktop and mobile views
Structured Sections

Header section with app branding and download links
Feature sections:
Channels
Messages
In Future
Footer section with download links and contact information
Modern UI Styling

CSS variables for consistent colors
Clean typography using the Montserrat font family
Balanced spacing and alignment using Flexbox
Reusable Layout

Alternating image and text layout for feature sections
Centralized content width using a common container
