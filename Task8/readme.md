# JavaScript Practice Project - Workspace Dashboard

## Project Overview
A complete web application with multiple tools built using JavaScript and Tailwind CSS. This project demonstrates practical JavaScript concepts through interactive features.

## Features Implemented

### 1. Carousel/Slider
- Next and previous buttons
- Auto-slide every 4 seconds
- Dot indicators for navigation
- Smooth transitions

### 2. Digital Clock
- Real-time display (HH:MM:SS)
- Updates every second
- Clean design at top of page

### 3. Calculator
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear button
- Backspace button

### 4. Password Generator
- Length selector (4-30 characters)
- Uppercase letters (A-Z)
- Lowercase letters (a-z)
- Numbers (0-9)
- Special characters (!@#$%^&*)
- Copy to clipboard button

### 5. Temperature Converter
- Celsius to Fahrenheit
- Fahrenheit to Celsius
- Real-time conversion

### 6. Task Manager (Bonus)
- Add new tasks
- Delete tasks
- Edit tasks
- Mark tasks as completed
- Local storage (tasks save after refresh)
- Pending tasks counter

## Technologies Used
- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Local Storage API
- Clipboard API
