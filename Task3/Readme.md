# Award Submission Form

A simple **HTML & CSS front-end form** for award submissions. Collects user details and award nominations in a two-column layout.

---

## How It Works

- Left section: Displays information and submission deadline.  
- Right section: Form with inputs for Name, Email, Organisation, and award checkboxes.  
- Each award has a textarea for short description (max 150 words).  
- Submit button confirms submission visually (no backend).

---

## Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling and layout  
- **Flexbox** – Side-by-side layout

---

## Terminologies

- **Flexbox** – Aligns sections horizontally  
- **Textarea** – Multi-line input  
- **Pseudo-element (::after)** – Decorative triangle in bottom-right  
- **Placeholder** – Hint text inside inputs  

---

## Notes

- Front-end only, no data storage.  
- Fully responsive and clean UI.  
- Suitable for assignments or portfolio demonstration.
